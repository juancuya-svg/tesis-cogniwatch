# Raw Data

Included in this thesis package:
- `DARWIN.csv` (small original dataset copied from the upload)
- `nacc_monitoring_sample.csv` (selected and reduced thesis-ready subset extracted from the larger uploaded NACC file)

The full NACC investigator CSV is intentionally not duplicated inside this package to keep the repository manageable.
The ML training scripts in this repo already use the included reduced monitoring subset for reproducibility.
