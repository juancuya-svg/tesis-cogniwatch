# Android Patient App

Android native patient app built with Kotlin + Jetpack Compose + Material 3.

## Core responsibilities
- Onboarding and consent.
- Caregiver linking.
- Health Connect permissions.
- Simple patient-facing summaries.
- Daily / weekly check-in.
- Optional cognitive-motor microtasks on the phone.
- Sync wearable summaries to backend.

## Open in Android Studio
Open this folder as a Gradle project.
