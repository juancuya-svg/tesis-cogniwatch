# ML and Analytics Layer

## Objective

Provide two complementary analytical components:

1. **Cognitive-motor proxy model** using DARWIN-derived features.
2. **Monitoring tier model** using selected NACC variables mapped to non-diagnostic support levels.

## Why two models instead of one

The uploaded datasets represent different phenomena:

- DARWIN captures **fine motor + cognitive proxy behavior** during digital writing tasks.
- NACC captures **clinical, functional, cognitive and longitudinal research observations**.

A single model would force incompatible assumptions. The correct design is a **hybrid engine**.
